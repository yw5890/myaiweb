## 컬러 사용 가이드

### Primary — `#c43e3c` 브릭 레드
- **사용처**: CTA 버튼, 링크, 섹션 구분 강조선, 카테고리 라벨
- **조합**: 흰색(`#ffffff`) 또는 세이지(`#eaf0d7`) 텍스트와 함께 사용
- **분위기**: 에디토리얼의 핵심 포인트 — 독자의 시선을 끄는 역할
- **예시**: "더 읽기", "구독하기", "프로젝트 보기" 버튼

### Background — `#eaf0d7` 세이지 그린
- **사용처**: 페이지 전체 배경, 섹션 배경, 사이드바
- **조합**: `#271914` 다크 텍스트와 함께 사용 시 최고 가독성
- **주의**: 텍스트 색상으로 사용 불가 (대비 부족) — 배경 전용

### Accent — `#e7be5a` 웜 골드
- **사용처**: 별점, 추천 뱃지, 헤더 강조선, 인용구 마커, 프리미엄 태그
- **조합**: `#271914` 다크 텍스트 위 배경으로 탁월 (고대비)
- **주의**: 소량 포인트 용도로만 (전체 면적 10% 이하)

### Dark — `#271914` 딥 다크 브라운
- **사용처**: 본문 텍스트, 헤더, 푸터 배경, 네비게이션 다크 테마
- **조합**: `#eaf0d7` (세이지) 또는 `#e7be5a` (골드) 텍스트와 함께 사용
- **분위기**: 고급스러운 인쇄물 느낌의 깊이감 부여

### Text Colors
| 변수 | Hex | 사용 상황 |
|------|-----|----------|
| `--color-text-primary` | `#271914` | 본문, 제목, 일반 텍스트 |
| `--color-text-secondary` | `#5c3020` | 서브 타이틀, 날짜, 저자명 |
| `--color-text-muted` | `#8a7060` | 캡션, 메타 정보, 태그 |
| `--color-text-on-dark` | `#eaf0d7` | 다크 배경(#271914) 위 |
| `--color-text-on-accent` | `#271914` | 골드 배경(#e7be5a) 위 |

## 반응형 고려사항

### 다크 모드 대응
```css
@media (prefers-color-scheme: dark) {
  :root {
    /* 배경: 다크 브라운 계열 심화 */
    --color-bg-primary:   #1a110d;  /* 더 깊은 다크 */
    --color-bg-secondary: #271914;  /* 원본 다크 컬러 → 서브 배경 */
    --color-bg-card:      #332418;  /* 카드 배경 */

    /* 텍스트: 세이지 & 골드로 전환 */
    --color-text-primary:   #eaf0d7;  /* 세이지 → 메인 텍스트 */
    --color-text-secondary: #c8d8b0;  /* 세이지 다크 */
    --color-text-muted:     #8a9878;  /* 뮤트 세이지 */

    /* Primary: 살짝 밝게 (다크 배경 대비 확보) */
    --color-primary:       #e05858;  /* 밝아진 레드 */
    --color-primary-light: #f07878;
    --color-primary-dark:  #c43e3c;  /* 원본값 → dark mode의 dark */

    /* 테두리: 불투명도 낮춤 */
    --color-border:        rgba(200, 216, 176, 0.2);
    --color-border-strong: rgba(196, 62, 60, 0.7);
  }
}
```

### 고대비 모드 (접근성)
```css
@media (prefers-contrast: high) {
  :root {
    --color-text-primary:   #000000;
    --color-button-primary: #8c0000;  /* 더 진한 레드 */
    --color-button-hover:   #600000;
    --color-border-strong:  #8c0000;
    --color-link:           #8c0000;
  }
}
```

### 모바일 최적화
```css

  .section-divider {

    height: 3px;
  }
}
```

## 색상 접근성 (WCAG 기준)

| 조합 | 대비비 | 등급 | 사용 가능 여부 |
|------|--------|------|--------------|
| `#271914` on `#eaf0d7` | 13.8:1 | AAA | 본문 텍스트 최적 |
| `#271914` on `#e7be5a` | 8.2:1  | AAA | 골드 배경 텍스트 최적 |
| `#ffffff` on `#c43e3c` | 4.6:1  | AA  | 버튼 텍스트 적합 |
| `#eaf0d7` on `#271914` | 13.8:1 | AAA | 다크 배경 텍스트 최적 |
| `#c43e3c` on `#eaf0d7` | 4.5:1  | AA  | 링크/강조 텍스트 적합 |
| `#e7be5a` on `#eaf0d7` | 2.1:1  | 미달 | 텍스트 사용 불가 (장식용만) |

> **주의**: `#e7be5a` 골드는 세이지 그린 배경 위에서 대비 부족 — 골드는 반드시 `#271914` 다크 배경 위에서 텍스트/장식으로 사용할 것.
